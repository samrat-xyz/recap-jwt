# JWT Recap Project (Express + MongoDB)

A simple backend recap project to practice **JWT Authentication & Authorization**
using **Express.js** and **MongoDB**.

---

## Tech Stack

- Node.js
- Express.js
- MongoDB (Native Driver)
- JSON Web Token (JWT)
- dotenv

---

## Project Purpose (What I Recapped)

- Express server setup
- MongoDB connection using MongoClient
- User registration
- User login
- JWT token creation
- JWT token verification middleware
- Protected route access
- Fetch logged-in user info securely
